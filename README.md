# TECHUP-Board Project
#### Techup GoFive team's Board Mini Project

**GoFive Board** 프로젝트는 정규 팀 프로젝트 전 개인 역량과 팀원들과의 소통을 위한 Mini Project 입니다.

## 주요기능

- 회원가입
  - Email 인증 및 중복체크
- 로그인
- 게시판 생성
- 게시글 생성 및 수정, 삭제
- 댓글 생성 및 수정, 삭제


## 기술 스택

| 영역                | 기술                  |
|-------------------|---------------------|
| Language          | Java 21             |
| Backend           | Spring Boot 3.5.7   |
| DB                | H2-In-memory        |
| Auth              | JWT (예정)            |
| DevOps            | X                   |
| Docs              | Swagger (Springdoc) |
| Source management | git, github         |
